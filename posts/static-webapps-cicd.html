Static Web Apps + CI/CD in 15 Minutes

2025-10-14 • 5 min read

Fork, configure a token, push to main, and let GitHub Actions publish automatically.

GitHub Action (copy & paste)

name: Azure Static Web Apps CI/CD

on:
  push:
    branches: [main]

jobs:
  build_and_deploy_job:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: Azure/static-web-apps-deploy@v1
        with:
          azure_static_web_apps_api_token: ${{ secrets.AZURE_STATIC_WEB_APPS_API_TOKEN }}
          repo_token: ${{ secrets.GITHUB_TOKEN }}
          action: upload
          app_location: /
          api_location: ""
          output_location: /

Steps

  1. Create Azure Static Web App and get the deployment token.
  2. Add repo secret AZURE_STATIC_WEB_APPS_API_TOKEN.
  3. Commit your static files to main.
  4. Push → Action runs → site publishes.
