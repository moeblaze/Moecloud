Playbook Implementation (Zero‑Trust Cloud)

Duration: 4–6 weeks · Format: blended (self‑paced + live labs) · Level: Intermediate‑Advanced

Outcomes

  • Design and deploy a Zero‑Trust landing zone with VPCs and secure networking
  • Stand up EKS with GitOps pipelines, policies (OPA/Gatekeeper), and observability
  • Implement serverless APIs (API Gateway + Lambda) and data layers (RDS/DynamoDB)
  • Ship a production‑grade edge (S3 + CloudFront + WAF) and event backbone (SQS/SNS/EventBridge)
  • Wire IAM SSO, Secrets Manager + KMS, and incident telemetry (Prometheus/Grafana/Otel)

Prerequisites

  • Cloud fundamentals (IAM, networking, CI/CD basics)
  • Git and container basics
  • Familiarity with YAML and Infrastructure‑as‑Code helpful

Curriculum

  1. Foundations & Landing Zone — accounts, org guardrails, VPCs, subnets, routing, zero‑trust design
  2. EKS + GitOps — cluster bootstrap, ArgoCD/Flux patterns, OPA/Gatekeeper, secrets, progressive delivery
  3. App & Data — API Gateway + Lambda, RDS schema patterns, DynamoDB for scale, ECR supply‑chain
  4. Edge & Events — S3 static hosting, CloudFront + WAF, SQS/SNS/EventBridge designs
  5. Observability & Resilience — Prometheus, Grafana, OpenTelemetry, SLOs, MTTR drills
  6. Security & Compliance — IAM SSO, RBAC, KMS envelope encryption, audit trails
  7. Capstone — deploy a reference workload end‑to‑end with CI/CD and guardrails

Labs

  • Lab 1: Landing Zone bootstrap (network + identity)
  • Lab 2: GitOps pipeline with policy checks
  • Lab 3: Serverless API with persistence
  • Lab 4: Edge hardening (WAF rules + CDN)
  • Lab 5: Observability with OTel traces
  • Lab 6: DR game day (chaos & recovery)
