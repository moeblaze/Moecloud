MBCC Sports Hydrator Switch
--------------------------------
This update points the hydrator to /data/sports-latest.json.

Deploy:
1) Upload/replace:
   /sports.html
   /assets/js/sports.js
   /data/sports-latest.json
2) Cache-bust:
   https://www.moecommunitycloud.com/sports.html?cb=latest
3) (Optional) Enable the GitHub Action scanner you downloaded earlier to auto-update /data/sports-latest.json.
